HTML
HTML là chữ viết tắt của cụm từ HyperText Markup Language (ngôn ngữ đánh dấu siêu văn bản)
CSS
CSS là chữ viết tắt của cụm từ Cascsding Style Sheet, nó là một ngôn ngữ quy định cách hiển thị của các thẻ html trên trang wed.
Javascript
JavaScript là một ngôn ngữ lập trình kịch bản dựa trên đối tượng được phát triển từ các ý niệm nguyên mẫu
